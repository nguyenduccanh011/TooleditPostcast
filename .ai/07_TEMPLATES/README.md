# Templates

Các template này dùng để tạo file docs mới cho dự án.

## Cách dùng
1. Copy template từ `07_TEMPLATES/` sang `docs/`
2. Đổi tên file (bỏ `.template`)
3. Điền nội dung theo template

## Templates có sẵn
- `README.template.md` -> `README.md` (root)
- `state.template.md` -> `docs/state.md`
- `active.template.md` -> `docs/active.md`
- `decisions.template.md` -> `docs/decisions.md`
- `issues.template.md` -> `docs/issues.md`
- `worklog.template.md` -> `docs/worklog.md`
- `code_rules.template.md` -> `docs/code_rules.md`
- `arch.template.md` -> `docs/arch.md`
- `db.template.md` -> `docs/db.md`
- `api.template.md` -> `docs/api.md`
- `archive.template.md` -> `docs/archive.md`
- `knowledge.template.md` -> `docs/knowledge/<topic>.md`

## Lưu ý
- Không sửa template gốc
- Chỉ copy và điền nội dung cho dự án cụ thể
- Tuân thủ format và giới hạn dòng (đặc biệt cho HOT docs)
